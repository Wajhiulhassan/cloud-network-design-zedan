This project was created as part of the HND in Digital Technologies – Unit 6: Networking in the Cloud assignment.

The aim of the project is to design and implement a hybrid cloud-based enterprise network that supports:

Cloud connectivity
Inter-department communication
Secure remote access
Wireless networking
DNS & DHCP services
Network scalability
Performance optimisation
Security enhancements using ACLs, NAT, PAT, and DMZ
The network infrastructure was simulated using Cisco Packet Tracer and tested for performance, latency, scalability, and security.

Business Scenario
The project is based on a case study for ZEDAN Enterprises, a Lahore-based company migrating from a traditional on-premises infrastructure to a cloud-enabled enterprise network with support from Stratosphere Networks.

The designed solution provides:
Centralised cloud communication
Secure access to cloud resources
Multi-office connectivity
Wireless and wired infrastructure
High availability and scalability
Enterprise-level security implementation

Performance Testing
The network was tested using Cisco Packet Tracer simulation tools.

Latency

Throughput
Bandwidth Utilisation
Results
Metric	Before Enhancement	After Enhancement
Average RTT	31.75 ms	30.25 ms
Latency	15.875 ms	15.125 ms
Jitter	28.67 ms	7.75 ms
Throughput	256 bps	312 bps
